# WHM Dashboard - Web-based CI/CD Monitoring Dashboard

## Project Overview
- **Project Name**: WHM Dashboard
- **Type**: Web Application (Express + React)
- **Core Functionality**: A web-based dashboard for monitoring multiple CI/CD pipelines in real-time
- **Target Users**: DevOps teams who want a visual overview of all their pipelines

## Features
- Real-time dashboard with multiple repositories
- Visual status indicators
- Historical trends and charts
- Health score visualization
- Dark/Light theme support
- Responsive design
- REST API for integration
